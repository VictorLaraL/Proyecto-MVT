Descripcion de MVT

Este procedimiento de buscar el bloque que mejor puede satisfacer la demanda de un proceso es un problema de la asignación dinámica de la memoria, el cual consiste en cómo cumplir una petición de tamaño N entre una lista de bloques disponibles.

Inicialmente, toda la memoria está disponible para los procesos de usuario y es considerada como un sólo bloque grande de memoria (un área libre o “hole”).

Cuando un proceso llega y necesita memoria se busca el bloque de memoria suficientemente grande para este proceso.
